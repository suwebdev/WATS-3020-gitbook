Quiz: Object Oriented JavaScript

Try this self-check quiz to test your knowledge!

What do we mean by "abstraction"?

Organizing code into functions, subroutines, and class/method definitions to better contain and isolate functionality and to mask the complexity of the code with an elegant interface. Replacing characters with encoded characters. Breaking code into small parts and spreading them everywhere. Writing code about complex poetry subjects.. Abstraction is the process of containing complex instructions and making them accessible through an elegant interface.

Inheriting features from another Class uses what command in JavaScript?

inherits extends overrides compares The command extends is used to inherit from another Class.

What term describes the concept of altering the functionality of inherited Class methods?

parameterization adjustment polymorphism structuring Polymorphism describes the concept of altering the functionality of inherited Class methods.

What is the method that initializes a Class called in JavaScript?

init() build() create() constructor() The constructor() method is called to initialize an instance of a Class.

The get command creates what kind of property on a Class?

dynamic writeable readable safe The get command creates a dynamic property on a Class.

Which command is used to bring in the existing functionality of a given method from the parent Class so that additional functionality can be added around it?

break() swing() super() include() The super() command is used to bring in existing functionality of a method from the parent Class.
